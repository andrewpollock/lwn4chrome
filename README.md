# Makes LWN slightly easier to read

This extension adds a title mouseover for all internal links to other articles on LWN.net. Know what article you're navigating to before you navigate to it.

Available in the Chrome Webstore as [LWN Helper](https://chromewebstore.google.com/detail/lwn-helper/dokopkjfamcinchhakehipepopknnlbf)
